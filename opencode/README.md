# OpenCode 配置文档整理

这份目录基于 `https://opencode.ai/docs/zh-cn` 中与“配置”相关的页面整理而成。

整理方式：
- 每个文件都保留两层内容：`理解` 和 `原文整理`。
- `理解` 是对模块职责、使用边界和配置重点的快速说明。
- `原文整理` 基于官方 zh-cn MDX 源文件做 Markdown 规范化，尽量保留原有层级、示例和说明文本。
- 每个文件顶部都保留了官方页面链接和源码来源，便于继续核对。
